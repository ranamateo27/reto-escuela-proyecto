# Proyecto Reto_Escuela

Sistema integral de gestión escolar con:

- Base de datos Oracle 23c (JSON Duality Views)
- API REST en Node.js / Express (15 endpoints)
- Natural Language Query (D9) para preguntas en español

## Inicio rápido

1. Clonar repo:

```bash
git clone https://github.com/ranamateo27/reto-escuela-proyecto.git
cd reto-escuela-proyecto
```

2. Revisar documentación principal:

```bash
# Lee el resumen
cat PROYECTO-FINAL.md
```

3. Extraer código (archivo `D6-Codigo-Fuente.md`) y crear estructura `src/` con los archivos indicados.

4. Instalar dependencias:

```bash
npm install
```

5. Configurar variables en `.env` (usar `.env.example` como referencia).

6. Ejecutar en desarrollo:

```bash
npm run dev
# o
node server.js
```

7. Abrir Swagger UI:

```
http://localhost:3000/api-docs
```

## Archivos importantes
